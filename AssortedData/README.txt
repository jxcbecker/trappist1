Data in this folder includes:

SWITFT_UVOT/  --- the SWIFT UV photometry, corrected for background flaring. 

manual_LC.csv --- the summary lightcurve of the SWIFT data, which comes from
		  each 'sum' file.

timescale_plot_data.pkl --- the pickle containing the tests of various k/Q
		  ratios for each planet in the TRAPPST-1 system. 

trappist_system_properties.csv --- the planetary properties for each planet
		  used in this work, mostly taken from Grimm+ 2018. 
		  These are inputs to the analysis.
